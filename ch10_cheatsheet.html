csv module — basics
SETUP Import & Sample File
import csv

# Sample file: data.csv
# name,age,city
# Alice,30,London
# Bob,25,Paris
Key: All values read from CSV are strings by default — cast explicitly if needed
reading csv
READER csv.reader — row as list
import csv

with open("data.csv", "r", newline="") as f:
    reader = csv.reader(f)

    header = next(reader)     # ['name', 'age', 'city']

    for row in reader:
        print(row)             # ['Alice', '30', 'London']
        print(row[0], row[1]) # Alice 30
newline="": Always pass newline="" when opening CSV files — prevents double newlines on Windows
DICTREADER csv.DictReader — row as dict
import csv

with open("data.csv", "r", newline="") as f:
    reader = csv.DictReader(f)

    for row in reader:
        print(row)
        # {'name': 'Alice', 'age': '30', 'city': 'London'}
        print(row["name"], row["age"])
        # Alice 30
DictReader vs reader: DictReader uses first row as keys automatically — much more readable; prefer this in production
READER Read all rows into a list
with open("data.csv", newline="") as f:
    reader = csv.DictReader(f)
    rows = list(reader)         # list of dicts

# Now use outside with block
print(rows[0]["name"])        # Alice
print(len(rows))               # 2
Key: Once outside with block, file is closed — convert to list inside the block if you need data later
writing csv
WRITER csv.writer — write lists
import csv

rows = [
    ["name", "age", "city"],
    ["Alice", 30, "London"],
    ["Bob",   25, "Paris"],
]

with open("output.csv", "w", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(rows[0])    # write single row
    writer.writerows(rows[1:])  # write multiple rows
DICTWRITER csv.DictWriter — write dicts
import csv

rows = [
    {"name": "Alice", "age": 30, "city": "London"},
    {"name": "Bob",   "age": 25, "city": "Paris"},
]

with open("output.csv", "w", newline="") as f:
    fieldnames = ["name", "age", "city"]
    writer = csv.DictWriter(f, fieldnames=fieldnames)

    writer.writeheader()         # writes: name,age,city
    writer.writerows(rows)       # writes all dicts
DictWriter — always call writeheader() first to write column names
custom delimiters & options
OPTIONS delimiter, quotechar, quoting
# TSV — tab-separated
reader = csv.reader(f, delimiter="\t")

# Semicolon-separated (common in European CSVs)
reader = csv.reader(f, delimiter=";")

# Custom quote character
reader = csv.reader(f, quotechar="'")

# Skip initial spaces after delimiter
reader = csv.reader(f, skipinitialspace=True)
Default delimiter is comma — change it for TSV or region-specific files
common patterns
PATTERN Filter rows & type casting
with open("data.csv", newline="") as f:
    reader = csv.DictReader(f)

    # Cast age to int, filter over 18
    adults = [row for row in reader
              if int(row["age"]) >= 18]
All CSV values are strings — always cast: int(row["age"]), float(row["price"])
PATTERN Append rows to existing CSV
with open("data.csv", "a", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["Charlie", 28, "Berlin"])

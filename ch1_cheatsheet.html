Chapter 1 — Quick Revision Cheat Sheet

variables
VAR Declaring Variables
name = "Alice"      # str
age  = 25           # int
pi   = 3.14         # float
flag = True         # bool
Note: No need to declare type — Python infers it automatically (dynamic typing)
VAR Multiple Assignment
x, y, z = 10, 20, 30   # assign different values
a = b = c = 100        # assign same value to all
VAR Naming Rules
Rule Example
Must start with letter or _ name, _name ✓
Cannot start with number 2name ✗
Case sensitive name ≠ Name ≠ NAME
No spaces — use underscore first_name ✓
Cannot be a keyword if, for, while ✗
TIP Dynamic Typing
x = 10         # x is int
x = "Python"   # x is now str — valid in Python
print(type(x)) # <class 'str'>
Key: Variable type can change at runtime — this is called dynamic typing

TYPE Truthy & Falsy Values
Falsy (treated as False) Truthy (treated as True)
0, 0.0, 0j Any non-zero number
"" (empty string) "hello", "False", "0"
[] (empty list) [0], [False]
None Any object
False True
bool("False") → True — non-empty string is always truthy
type casting
CAST Casting Functions
int(10.9)       # 10     — truncates, does NOT round
int("25")       # 25     — string to int
float(10)       # 10.0   — int to float
str(499)        # "499"  — int to string
bool(0)         # False
bool("hello")   # True
Key: int() truncates — int(10.9) → 10, NOT 11
CAST Common Casting Errors
# ✗ Wrong — cannot concat str + int
age = 25
print("Age is " + age)        # TypeError

# ✓ Correct
print("Age is " + str(age))    # Age is 25
print(f"Age is {age}")          # Age is 25
print()
PRINT All Print Styles
name = "Alice"
age  = 25

print(name, age)                  # Alice 25
print("Name:", name)             # Name: Alice
print("Name: " + name)           # Name: Alice (concat)
print(f"Name: {name}, Age: {age}") # f-string (recommended)
PRINT sep parameter
print("a", "b", "c")
# a b c  (default sep=" ")

print("a", "b", "c", sep="-")
# a-b-c

print("a", "b", sep="")
# ab
PRINT end parameter
print("Hello")
print("World")
# Hello
# World  (default end="\n")

print("Hello", end=" ")
print("World")
# Hello World
PRINT f-strings (Most Used)
name = "Alice"
score = 95.678

print(f"Hello {name}")              # Hello Alice
print(f"Score: {score:.2f}")        # Score: 95.68
print(f"{name} scored {score:.0f}")  # Alice scored 96
print(f"2 + 2 = {2+2}")            # 2 + 2 = 4  (expression inside)

NUM / vs //
x = 5
y = 2

print(x /  y)  # 2.5  → float
print(x // y)  # 2    → floor
print(x %  y)  # 1    → remainder
print(x ** y)  # 25   → power
NUM String * Number
x = "50"
print(x * 3)   # "505050"
# string repetition, not math!

x = "Ha"
print(x * 3)   # "HaHaHa"
Trap: "50" * 3 = "505050", not 150
NUM Number Functions
abs(-5)          # 5      — absolute value
round(3.567, 2)   # 3.57   — rounds to 2 decimal places
round(2.5)        # 2      — banker's rounding (rounds to even!)
round(3.5)        # 4
pow(2, 10)        # 1024   — same as 2**10
max(3, 7, 1)      # 7
min(3, 7, 1)      # 1
round(2.5) → 2, NOT 3 — Python uses banker's rounding (round half to even)
NUM math module
import math

math.sqrt(16)     # 4.0
math.ceil(4.2)    # 5   — rounds UP always
math.floor(4.9)   # 4   — rounds DOWN always
math.pi           # 3.141592653589793
math.factorial(5) # 120

Python → Loops & range()

Chapter 3 — Quick Revision Cheat Sheet

for loop
FOR Basic Syntax
for i in range(5):
    print(i)              # 0 1 2 3 4

for item in ["a", "b", "c"]:
    print(item)           # a  b  c

for char in "Python":
    print(char)           # P y t h o n
FOR enumerate() — index + value
for i, val in enumerate(["a", "b", "c"]):
    print(i, val)
# 0 a  /  1 b  /  2 c

# Start index from 1
for i, val in enumerate(["a", "b"], start=1):
    print(i, val)
# 1 a  /  2 b
Always prefer enumerate() over range(len(list)) when you need index + value
FOR else block
for i in range(3):
    print(i)
else:
    print("Done")    # runs after loop completes normally

# Real world use — search pattern
for item in data:
    if item == target:
        print("Found")
        break
else:
    print("Not found")   # only runs if break never hit
Key: else is SKIPPED if loop exits via break — runs even on empty range
while loop
WHILE Basic Syntax
i = 0
while i < 5:
    print(i)
    i += 1          # 0 1 2 3 4
Always update loop variable — else infinite loop
WHILE Infinite loop + break
while True:
    user_input = input("Enter: ")
    if user_input == "quit":
        break
Real World Use: CLI menus, retry logic, polling, event loops
range()
RANGE All 3 Forms
SyntaxOutputNotes
range(5)0, 1, 2, 3, 4start=0, step=1 (defaults)

Stop value is ALWAYS excluded
RANGE Common Patterns
list(range(5))           # [0, 1, 2, 3, 4]
list(range(1, 6))        # [1, 2, 3, 4, 5]
list(range(1, 10, 2))    # [1, 3, 5, 7, 9]   → odd
list(range(0, 11, 2))    # [0, 2, 4, 6, 8, 10] → even
list(range(10, 0, -2))   # [10, 8, 6, 4, 2]

# range is lazy — doesn't store all values in memory
len(range(0, 100))        # 100 — supports len()
range(10)[3]              # 3  — supports indexing
break / continue / pass
BREAK Exit immediately
for i in range(10):
    if i == 5:
        break
    print(i)
# 0 1 2 3 4
Key: else block is SKIPPED
CONTINUE Skip iteration
for i in range(5):
    if i == 2:
        continue
    print(i)
# 0 1 3 4
Key: else block still runs
PASS Placeholder — does nothing
for i in range(5):
    pass              # valid empty loop — no error
pass vs continue: pass does nothing and continues rest of body; continue jumps to next iteration immediately
nested loops
NESTED Basic Pattern
for i in range(3):
    for j in range(3):
        print(i, j)
# (0,0)(0,1)(0,2)(1,0)(1,1)(1,2)(2,0)(2,1)(2,2) → 9 total
Rule: break only exits the innermost loop — outer loop continues
NESTED Break outer loop using flag
found = False
for i in range(5):
    for j in range(5):
        if i * j == 6:
            found = True
            break
    if found:
        break

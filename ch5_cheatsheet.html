basics
LIST Creating Lists
nums    = [1, 2, 3, 4, 5]

Key: Lists are ordered, mutable, and allow duplicates
indexing
INDEX Positive & Negative Indexing
nums = [10, 20, 30, 40, 50]
#       0    1    2    3    4   ← positive
#      -5   -4   -3   -2   -1  ← negative

print(nums[0])    # 10  — first
print(nums[2])    # 30
print(nums[-1])   # 50  — last
print(nums[-2])   # 40  — second from last
slicing
SLICE Syntax: list[start:stop:step]
nums = [10, 20, 30, 40, 50]

nums[1:4]      # [20, 30, 40]   — stop excluded
nums[:3]       # [10, 20, 30]   — from start
nums[2:]       # [30, 40, 50]   — to end
nums[:]        # [10, 20, 30, 40, 50] — full copy
nums[::2]      # [10, 30, 50]   — every 2nd
nums[::-1]     # [50, 40, 30, 20, 10] — reversed
nums[1:4:2]    # [20, 40]
Key: Slicing always returns a NEW list — original is unchanged
list methods
ADD Adding Elements
fruits = ["apple", "banana"]

fruits.append("mango")          # adds to END → ["apple","banana","mango"]
fruits.insert(1, "cherry")       # inserts at index 1
fruits.extend(["grape", "kiwi"])  # adds multiple items from iterable
append vs extend: append([1,2]) adds a list as one element; extend([1,2]) adds each element individually
REMOVE Removing Elements
nums = [1, 2, 3, 2, 4]

nums.remove(2)      # removes FIRST occurrence of 2 → [1,3,2,4]
nums.pop()          # removes & returns LAST element
nums.pop(1)         # removes & returns element at index 1
del nums[0]         # deletes element at index 0
nums.clear()        # empties the list → []
SEARCH Finding Elements
nums = [10, 20, 30, 20]

nums.index(20)      # 1 — index of FIRST occurrence
nums.count(20)      # 2 — how many times 20 appears
20 in nums           # True
99 in nums           # False
SORT Sorting & Reversing
nums = [3, 1, 4, 1, 5]

nums.sort()                    # sorts IN-PLACE ascending → [1,1,3,4,5]
nums.sort(reverse=True)       # sorts IN-PLACE descending
nums.reverse()                 # reverses IN-PLACE

# sorted() — returns NEW list, original unchanged
new = sorted(nums)
new = sorted(nums, reverse=True)
new = sorted(words, key=len)    # sort by length
sort() vs sorted(): sort() modifies in-place, returns None; sorted() returns new list, original unchanged
COPY Copying a List
a = [1, 2, 3]

b = a              # NOT a copy — both point to same list
b = a.copy()       # shallow copy ✓
b = a[:]           # shallow copy via slice ✓
b = list(a)        # shallow copy via constructor ✓
b = a does NOT copy — modifying b also modifies a
OTHER Other Useful Methods
nums = [3, 1, 4]

len(nums)           # 3 — number of elements
sum(nums)           # 8
min(nums)           # 1
max(nums)           # 4
nums + [5, 6]       # [3,1,4,5,6] — concatenation → new list
nums * 2            # [3,1,4,3,1,4] — repetition
list comprehension
COMP Syntax & Examples
# Syntax: [expression  for item in iterable  if condition]

squares  = [x**2 for x in range(5)]         # [0,1,4,9,16]
evens    = [x for x in range(10) if x%2==0]  # [0,2,4,6,8]
upper    = [w.upper() for w in ["hi", "bye"]] # ['HI','BYE']
flat     = [x for row in [[1,2],[3,4]] for x in row]  # [1,2,3,4]
Faster and more Pythonic than loop + append for building lists
nested lists
NEST Accessing Nested Elements
matrix = [[1, 2, 3],
          [4, 5, 6],
          [7, 8, 9]]

matrix[0][0]    # 1 — row 0, col 0
matrix[1][2]    # 6 — row 1, col 2
matrix[-1][-1]  # 9 — last row, last col

# Flatten nested list
flat = [x for row in matrix for x in row]
# [1,2,3,4,5,6,7,8,9]

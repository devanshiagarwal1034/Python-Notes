Chapter 7 — Quick Revision Cheat Sheet

sets — basics
SET Creating Sets
nums   = {1, 2, 3, 4}
fruits = {"apple", "banana", "mango"}
empty  = set()           # NOT {} — that creates a dict!

# From list — removes duplicates
unique = set([1, 2, 2, 3, 3])   # {1, 2, 3}

print(type(nums))   # <class 'set'>
Key: Sets are unordered, mutable, no duplicates, and elements must be hashable
SET Add / Remove
s = {1, 2, 3}

s.add(4)           # {1,2,3,4} — adds one element
s.add(2)           # no change — duplicates ignored silently
s.update([5, 6])  # adds multiple elements from iterable

s.remove(3)       # removes 3 — KeyError if missing
s.discard(99)     # removes 99 — NO error if missing
s.pop()           # removes & returns an ARBITRARY element
s.clear()         # empties the set
remove vs discard: remove() raises KeyError if missing; discard() silently ignores
SET Membership & Length
s = {1, 2, 3}

2 in s       # True  — O(1) lookup
9 in s       # False
len(s)       # 3
Set membership check is O(1) — much faster than list's O(n)
set operations
OPS All Set Operations
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Union — all elements from both
a | b                # {1,2,3,4,5,6}
a.union(b)          # same

# Intersection — only common elements
a & b                # {3,4}
a.intersection(b)   # same

# Difference — in a but NOT in b
a - b                # {1,2}
a.difference(b)     # same

# Symmetric difference — in either but NOT both
a ^ b                             # {1,2,5,6}
a.symmetric_difference(b)        # same
SUBSET Subset / Superset / Disjoint
a = {1, 2}
b = {1, 2, 3, 4}

a.issubset(b)     # True  — all of a is in b
b.issuperset(a)   # True  — b contains all of a
a <= b            # True  — subset operator
a <  b            # True  — proper subset (a != b)

{1,2}.isdisjoint({3,4})   # True  — no common elements
{1,2}.isdisjoint({2,3})   # False — 2 is common
FROZEN frozenset — immutable set
fs = frozenset([1, 2, 3])

fs.add(4)     # AttributeError — immutable!

# Use as dict key or set element
d = {frozenset({1,2}): "pair"}  # valid — frozenset is hashable
Use frozenset when: you need a set as a dict key or inside another set
tuples — basics
TUPLE Creating Tuples
coords  = (10, 20)
single  = (42,)         # trailing comma required for single element
empty   = ()
no_paren = 1, 2, 3      # parentheses optional — still a tuple

print(type((42,)))   # <class 'tuple'>
print(type((42)))    # <class 'int'> — NOT a tuple!
Key: Tuples are ordered, immutable, allow duplicates — faster than lists
TUPLE Indexing & Slicing
t = (10, 20, 30, 40)

t[0]      # 10
t[-1]     # 40
t[1:3]    # (20, 30)
t[::-1]   # (40, 30, 20, 10)

t.index(20)   # 1 — find index of value
t.count(20)   # 1 — count occurrences
tuple unpacking
UNPACK All Unpacking Patterns
# Basic unpacking
x, y = (10, 20)          # x=10, y=20

# Swap variables
a, b = b, a               # Pythonic swap — no temp variable

# Extended unpacking with *
first, *rest = [1, 2, 3, 4]   # first=1, rest=[2,3,4]
*init, last  = [1, 2, 3, 4]   # init=[1,2,3], last=4
a, *b, c     = [1, 2, 3, 4]   # a=1, b=[2,3], c=4

# Ignore values with _
x, _, z = (1, 2, 3)           # _ is convention for "don't care"
UNPACK Unpacking in loops
pairs = [(1, "one"), (2, "two"), (3, "three")]

for num, word in pairs:
    print(num, word)

# Works with dict.items() too
for key, val in d.items():
    print(key, val)

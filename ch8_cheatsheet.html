lambda — basics
LAMBDA Syntax & Examples
# Syntax: lambda parameters : expression

# Equivalent def vs lambda
def square(x):
    return x ** 2

square = lambda x: x ** 2
print(square(5))          # 25

# Multiple parameters
add  = lambda a, b: a + b
print(add(3, 4))           # 7

# With default argument
greet = lambda name, msg="Hello": f"{msg}, {name}"
print(greet("Bob"))        # Hello, Bob

# No parameters
pi = lambda: 3.14159
print(pi())                 # 3.14159
Key: Lambda can only have ONE expression — no statements, no assignments, no loops
LAMBDA With conditional (ternary)
classify = lambda x: "even" if x % 2 == 0 else "odd"
print(classify(4))    # even
print(classify(7))    # odd

larger = lambda a, b: a if a > b else b
print(larger(3, 7))   # 7
map()
MAP Apply function to every element
# Syntax: map(function, iterable)
# Returns a map object — convert to list to see results

nums = [1, 2, 3, 4, 5]

squares = list(map(lambda x: x**2, nums))
# [1, 4, 9, 16, 25]

doubled = list(map(lambda x: x*2, nums))
# [2, 4, 6, 8, 10]

# With built-in function
words  = ["hello", "world"]
upper  = list(map(str.upper, words))
# ['HELLO', 'WORLD']

# map() with two iterables
a = [1, 2, 3]
b = [10, 20, 30]
sums = list(map(lambda x, y: x + y, a, b))
# [11, 22, 33]
map() vs list comprehension: both work; list comprehension is generally more readable and Pythonic
filter()
FILTER Keep elements where function returns True
# Syntax: filter(function, iterable)
# Returns a filter object — convert to list

nums = [1, 2, 3, 4, 5, 6]

evens = list(filter(lambda x: x % 2 == 0, nums))
# [2, 4, 6]

positives = list(filter(lambda x: x > 0, [-2, -1, 0, 1, 2]))
# [1, 2]

# filter(None, iterable) — removes falsy values
cleaned = list(filter(None, [0, 1, "", "hi", None, 2]))
# [1, "hi", 2]
filter(None, iterable) removes all falsy values — handy for cleaning lists
sorted() with key
SORT Custom sort using lambda
words = ["banana", "kiwi", "apple", "mango"]

# Sort by length
sorted(words, key=lambda x: len(x))
# ['kiwi', 'apple', 'mango', 'banana']

# Sort list of tuples by second element
pairs = [("b", 2), ("a", 3), ("c", 1)]
sorted(pairs, key=lambda x: x[1])
# [('c',1), ('b',2), ('a',3)]

# Sort list of dicts by a key
people = [{"name": "Bob", "age": 30},
          {"name": "Alice", "age": 25}]
sorted(people, key=lambda x: x["age"])
# [{'name':'Alice','age':25}, {'name':'Bob','age':30}]

# Sort descending
sorted(words, key=lambda x: len(x), reverse=True)
reduce()
REDUCE Accumulate to single value
from functools import reduce

nums = [1, 2, 3, 4, 5]

# Sum all elements
total = reduce(lambda acc, x: acc + x, nums)
# 15  →  ((((1+2)+3)+4)+5)

# Find maximum
maximum = reduce(lambda a, b: a if a > b else b, nums)
# 5

# With initial value
total = reduce(lambda acc, x: acc + x, nums, 100)
# 115  →  starts accumulating from 100
reduce() needs: from functools import reduce — not built-in since Python 3
lambda vs def
COMPARE When to use which
Lambdadef
Single expression onlyMultiple statements allowed
No name (anonymous)Has a name
No docstringCan have docstring
Used inline (map, filter, sorted)Reusable, testable
Can't contain loops or assignmentsFull Python logic
Use for: throwaway key functionsUse for: anything complex or reused
Avoid assigning lambda to a variable — use def instead: `square = lambda x: x**2` is worse than `def square(x): return x**2`

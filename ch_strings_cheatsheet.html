indexing & slicing
INDEX Indexing & Slicing
s = "Python"
#    0 1 2 3 4 5   ← positive
#   -6-5-4-3-2-1   ← negative

s[0]      # 'P'
s[-1]     # 'n'
s[1:4]    # 'yth'
s[:3]     # 'Pyt'
s[3:]     # 'hon'
s[::-1]   # 'nohtyP'  — reversed
s[::2]    # 'Pto'     — every 2nd char
Key: Strings support all the same slicing as lists — and always return a new string
case methods
CASE All Case Methods
s = "hello world"

s.upper()        # "HELLO WORLD"
s.lower()        # "hello world"
s.title()        # "Hello World"   — each word capitalised
s.capitalize()   # "Hello world"   — only first char
s.swapcase()     # "HELLO WORLD" → "hello world"
strip & clean
STRIP strip / lstrip / rstrip
s = "  hello world  "

s.strip()     # "hello world"   — both sides
s.lstrip()    # "hello world  " — left only
s.rstrip()    # "  hello world" — right only

# Strip specific characters
"##hello##".strip("#")    # "hello"
"xxhelloxx".strip("x")    # "hello"
Real World Use: Cleaning user input, stripping newlines from file lines
find, replace & check
FIND find / index / count / replace
s = "hello world hello"

s.find("hello")       # 0    — first occurrence index
s.find("xyz")         # -1   — returns -1 if not found
s.index("world")      # 6    — like find but raises ValueError
s.rfind("hello")      # 12   — last occurrence
s.count("hello")      # 2    — how many times
s.replace("hello", "hi")       # "hi world hi"
s.replace("hello", "hi", 1)    # "hi world hello" — max 1 replace
find vs index: find() returns -1 if not found; index() raises ValueError — use find() when unsure
CHECK startswith / endswith / in
s = "hello world"

s.startswith("hello")   # True
s.endswith("world")     # True
"world" in s             # True
"xyz" in s               # False

# Check multiple prefixes/suffixes
s.startswith(("hello", "hi"))   # True — tuple of options
s.endswith((".txt", ".csv"))    # False
split & join
SPLIT split() — string to list
s = "apple,banana,mango"

s.split(",")          # ['apple', 'banana', 'mango']
s.split(",", 1)       # ['apple', 'banana,mango'] — max 1 split

"hello world".split()  # ['hello', 'world'] — splits on whitespace
"a  b  c".split()      # ['a', 'b', 'c'] — multiple spaces handled

# Split lines
"line1\nline2\nline3".splitlines()
# ['line1', 'line2', 'line3']
JOIN join() — list to string
words = ["apple", "banana", "mango"]

",".join(words)     # "apple,banana,mango"
" ".join(words)    # "apple banana mango"
"".join(words)     # "applebananamango"
" | ".join(words)  # "apple | banana | mango"

# Join characters back
"".join(["P", "y", "t"])   # "Pyt"
Key: join() is called on the separator, not the list — separator.join(list)
string formatting
FORMAT All Formatting Styles
name  = "Bob"
score = 95.678
num   = 1000000

# f-string (Python 3.6+) — recommended
f"Hello {name}, score: {score:.2f}"   # "Hello Bob, score: 95.68"
f"{num:,}"                            # "1,000,000"
f"{score:.0f}"                         # "96"
f"{name:>10}"                          # "       Bob"  right-align
f"{name:<10}"                          # "Bob       "  left-align
f"{name:^10}"                          # "   Bob    "  center
f"{num:0>8}"                           # "00001000"    zero-pad

# .format() style
"Hello {}, score: {:.2f}".format(name, score)

# % style (old)
"Hello %s, score: %.2f" % (name, score)
Always prefer f-strings — fastest, most readable, supports expressions

other useful methods
OTHER zfill, center, ljust, rjust
"42".zfill(5)          # "00042"  — zero pad to width 5
"hi".center(10)        # "    hi    "
"hi".center(10, "*")   # "****hi****"
"hi".ljust(10, "-")    # "hi--------"
"hi".rjust(10, "-")    # "--------hi"
OTHER encode / decode & ord / chr
# Encode string to bytes
"hello".encode("utf-8")     # b'hello'
b"hello".decode("utf-8")   # "hello"

# Character ↔ ASCII
ord("A")    # 65  — char to ASCII code
chr(65)     # 'A' — ASCII code to char
CONCAT Concatenation & Repetition
"Hello" + " " + "World"   # "Hello World"
"ha" * 3                    # "hahaha"

# Fast concatenation of many strings — use join()
parts = ["Hello", "World", "!"]
result = " ".join(parts)      # "Hello World !"
Never use + in a loop to build strings — O(n²); use join() or list then join at end
